Matching rows from both tables

All rows from left table

All rows from right table

FULL OUTER JOIN

All rows from both tables

Advanced SQL Examples 
-- JOIN Example
SELECT 
    o.order_id,
    c.customer_name,
    o.product,
    o.quantity,
    o.total_price
FROM orders o
INNER JOIN customers c ON o.customer_id = c.customer_id
WHERE o.order_date >= '2024-01-01';
-- Subquery Example
SELECT customer_name, total_spent
FROM (
    SELECT 
        customer_id,
        SUM(total_price) AS total_spent
    FROM orders
    GROUP BY customer_id
) customer_totals
WHERE total_spent > 1000;
-- VIEW Creation
CREATE VIEW monthly_sales AS
SELECT
    DATE_FORMAT(order_date, '%Y-%m') AS month,
    COUNT(*) AS order_count,
    SUM(total_price) AS total_revenue
FROM orders
GROUP BY DATE_FORMAT(order_date, '%Y-%m');
-- TRIGGER Example
CREATE TRIGGER update_inventory
AFTER INSERT ON orders
FOR EACH ROW
BEGIN
    UPDATE products
    SET stock_quantity = stock_quantity - NEW.quantity
    WHERE product_id = NEW.product_id;
END;

SQL Join Comparison

Join Type Description Use Case Visual Representation
INNER JOIN Returns only matching rows Find related data ∩ (Intersection)
LEFT JOIN All rows from left table + matches Include all primary records All Left + Matching Right
RIGHT JOIN All rows from right table + matches Include all foreign records All Right + Matching Left
FULL OUTER JOIN All rows from both tables Complete data merge ∪ (Union)
CROSS JOIN Cartesian product All combinations × (Multiplication)

DAX (Data Analysis Expressions) Basics

-- Basic DAX Functions
Total Sales = SUM(Sales[Amount])

Sales YTD = TOTALYTD(SUM(Sales[Amount]), 'Date'[Date])

Sales Previous Year = 
CALCULATE(
    SUM(Sales[Amount]),
    SAMEPERIODLASTYEAR('Date'[Date])
)

Profit Margin = 
DIVIDE(
    SUM(Sales[Profit]),
    SUM(Sales[Amount]),
    0
)

)

Moving Average 30 Days = 
AVERAGEX(
    DATESINPERIOD(
        'Date'[Date],
        LASTDATE('Date'[Date]),
        -30,
        DAY
    ),
    [Total Sales]
)

    TRUE(),
