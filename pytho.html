Hour 1 Variables, Data Types, Input/Output

✓

Why Python for AI/ML?

Python is the most popular language for AI and Machine Learning because of its simple syntax and powerful libraries.

What are Variables?

Variables are containers that store data values. In Python, you don't need to declare the type - Python figures it out automatically.

Key Concept: Dynamic Typing

Python is dynamically typed, meaning you don't need to specify data types. The interpreter figures it out at runtime.

Example: Creating Variables 
# String variable - for text
name = "John"
print(name)  # Output: John

# Integer variable - for whole numbers
age = 20
print(age)   # Output: 20

# Float variable - for decimal numbers
height = 5.8
print(height)  # Output: 5.8

# Boolean variable - True/False
is_student = True
print(is_student)  # Output: True

# Multiple assignment
x, y, z = 10, 20, 30
print(x, y, z)  # Output: 10 20 30

Data Types in Python

Data Type Example Description Use in AI/ML
int 10, -5, 0 Whole numbers (no decimals) Counting, indices
float 3.14, -2.5 Decimal numbers ML model weights
str "Hello", 'Python' Text (string of characters) NLP, text data
bool True, False Boolean values Conditions, flags

Hour 2 Lists and Tuples

75%

Data Structures for AI

Lists and tuples are essential for organizing data in AI projects - datasets, model parameters, results.

Lists vs Tuples

Lists (Mutable)

  • Can be changed after creation
  • Use square brackets []
  • Ideal for data that changes
  • AI Use: Training data, model predictions
# Creating a list
fruits = ["apple", "banana", "mango"]
print(f"Original: {fruits}")

# Add item
fruits.append("orange")
print(f"After append: {fruits}")

# Remove item
fruits.remove("banana")
print(f"After remove: {fruits}")

# Access elements
print(f"First fruit: {fruits[0]}")
print(f"Last fruit: {fruits[-1]}")

# List length
print(f"Total fruits: {len(fruits)}")

Tuples (Immutable)

  • Cannot be changed after creation
  • Use parentheses ()
  • Ideal for fixed data
  • AI Use: Model hyperparameters, constants
# Creating a tuple
coordinates = (10, 20, 30)
print(f"Coordinates: {coordinates}")

# Access elements (same as lists)
print(f"X coordinate: {coordinates[0]}")
print(f"Y coordinate: {coordinates[1]}")

# This will cause ERROR - tuples are immutable
# coordinates[0] = 15  # ERROR!

# Tuple unpacking
x, y, z = coordinates
print(f"X={x}, Y={y}, Z={z}")

# Single element tuple (comma is required)
single = (42,)
print(f"Single: {single}")

List Operations for AI

Common List Operations 
# AI Dataset example
dataset = [85, 90, 78, 92, 88, 76, 95]

# Basic statistics (like in ML)
max_score = max(dataset)
min_score = min(dataset)
avg_score = sum(dataset) / len(dataset)

print(f"Dataset: {dataset}")
print(f"Maximum: {max_score}")
print(f"Minimum: {min_score}")
print(f"Average: {avg_score:.2f}")

# List comprehension (Pythonic way)
squared_scores = [score**2 for score in dataset]
print(f"Squared scores: {squared_scores}")

# Filtering (like data preprocessing)
passed = [score for score in dataset if score >= 80]
print(f"Passed scores: {passed}")

Hour 3 Sets and Dictionaries

50%

Advanced Data Structures

Sets and dictionaries are crucial for handling unique data and key-value pairs in AI applications.

Sets (Unique Elements)

  • No duplicate elements
  • Unordered collection
  • Use curly braces {}
  • AI Use: Unique labels, vocabulary
# Creating a set
numbers = {1, 2, 3, 3, 4, 4, 5}
print(f"Set (duplicates removed): {numbers}")

# Set operations
set_a = {1, 2, 3, 4, 5}
set_b = {4, 5, 6, 7, 8}

print(f"Set A: {set_a}")
print(f"Set B: {set_b}")
print(f"Union (A ∪ B): {set_a.union(set_b)}")
print(f"Intersection (A ∩ B): {set_a.intersection(set_b)}")
print(f"Difference (A - B): {set_a.difference(set_b)}")

# AI Application: Unique words in text
text = "apple banana apple orange banana mango"
words = set(text.split())
print(f"\nUnique words: {words}")
print(f"Vocabulary size: {len(words)}")

Dictionaries (Key-Value Pairs)

  • Key-value pairs
  • Fast lookups
  • Use curly braces with colons {key: value}
  • AI Use: Model configurations, feature mapping
# Creating a dictionary
student = {
    "name": "Amit Sharma",
    "roll_no": 101,
    "course": "BCA",
    "subjects": ["Python", "ML", "Statistics"],
    "cgpa": 8.5
}

print(f"Student: {student}")
print(f"Name: {student['name']}")
print(f"Course: {student.get('course')}")

# Add new key
student["year"] = 3
print(f"With year: {student}")

# Update value
student["cgpa"] = 8.7
print(f"Updated CGPA: {student['cgpa']}")

# Dictionary methods
print(f"\nKeys: {student.keys()}")
print(f"Values: {student.values()}")
print(f"Items: {student.items()}")

# AI Application: Model configuration
model_config = {
    "learning_rate": 0.001,
    "batch_size": 32,
    "epochs": 100,
    "optimizer": "adam"
}
print(f"\nModel Config: {model_config}")

Dictionary Comprehension

Pythonic Dictionary Creation 
# Create dictionary from lists (common in AI)
features = ["age", "income", "education", "experience"]
values = [25, 50000, 16, 3]

# Using zip
data_dict = dict(zip(features, values))
print(f"Data Dictionary: {data_dict}")

# Dictionary comprehension
squares = {x: x**2 for x in range(1, 6)}
print(f"Squares: {squares}")

# Conditional dictionary comprehension
even_squares = {x: x**2 for x in range(1, 11) if x % 2 == 0}
print(f"Even Squares: {even_squares}")

# Nested dictionaries (common in complex AI models)
ai_model = {
    "model": {
        "name": "Neural Network",
        "layers": 3,
        "activation": "relu"
    },
    "training": {
        "data_size": 10000,
        "test_split": 0.2
    }
}
print(f"\nAI Model Structure: {ai_model}")

Control Flow in AI

Conditionals and loops are fundamental for implementing AI algorithms and data processing pipelines.

Conditional Statements (if-elif-else)

Decision Making in AI 
# AI Model Performance Evaluation
accuracy = 85.5

if accuracy >= 90:
    grade = "Excellent"
    status = "Production Ready"
elif accuracy >= 80:
    grade = "Good"
    status = "Needs Fine-tuning"
elif accuracy >= 70:
    grade = "Average"
    status = "Needs Improvement"
else:
    grade = "Poor"
    status = "Retrain Required"

print(f"Model Accuracy: {accuracy}%")
print(f"Grade: {grade}")
print(f"Status: {status}")

# Multiple conditions (common in AI rules)
temperature = 25
humidity = 60

if temperature > 30 and humidity > 70:
    weather = "Hot and Humid"
elif temperature < 10 or humidity < 30:
    weather = "Cold and Dry"
else:
    weather = "Pleasant"

print(f"\nTemperature: {temperature}°C, Humidity: {humidity}%")
print(f"Weather: {weather}")

Loops in Python

For Loops

Iterate over sequences (lists, tuples, strings)

# Iterate through list
ai_topics = ["Neural Networks", "Computer Vision", "NLP", "Reinforcement Learning"]

print("AI Topics:")
for topic in ai_topics:
    print(f"- {topic}")

# With enumerate (get index)
print("\nTopics with index:")
for idx, topic in enumerate(ai_topics, 1):
    print(f"{idx}. {topic}")

# Range function
print("\nTraining epochs:")
for epoch in range(1, 6):
    print(f"Epoch {epoch}: Training...")
    
# Loop through dictionary
model_params = {"lr": 0.001, "batch": 32, "epochs": 100}
print("\nModel Parameters:")
for key, value in model_params.items():
    print(f"{key}: {value}")

While Loops

Execute while condition is true

# AI Training simulation
epoch = 1
target_accuracy = 95
current_accuracy = 75

print("Training Model...")
while current_accuracy < target_accuracy:
    print(f"Epoch {epoch}: Accuracy = {current_accuracy}%")
    current_accuracy += 5  # Simulate improvement
    epoch += 1
    
    if epoch > 10:  # Safety check
        print("Maximum epochs reached!")
        break

print(f"\nTraining completed!")
print(f"Final Accuracy: {current_accuracy}%")
print(f"Total Epochs: {epoch-1}")

# Infinite loop with break (common in AI training)
data_points = [10, 20, 30, 40, 50]
idx = 0
while True:
    if idx >= len(data_points):
        print("\nAll data processed!")
        break
    print(f"Processing data point {data_points[idx]}...")
    idx += 1

Complete Program Examples

Real AI-inspired Programs 
# Program 1: Student Grade Calculator (Like ML Classification)
def calculate_grade(marks):
    if marks >= 90:
        return "A+"
    elif marks >= 80:
        return "A"
    elif marks >= 70:
        return "B"
    elif marks >= 60:
        return "C"
    else:
        return "F"

# Test the function
students = [("Amit", 85), ("Priya", 92), ("Rohan", 78), ("Ananya", 65)]
print("Student Grades:")
for name, marks in students:
    grade = calculate_grade(marks)
    print(f"{name}: {marks} → {grade}")

# Program 2: Data Analyzer (Like Simple ML)
data = [23, 45, 67, 89, 12, 34, 56, 78, 90, 21]
print(f"\nData Analysis:")
print(f"Dataset: {data}")
print(f"Count: {len(data)}")
print(f"Sum: {sum(data)}")
print(f"Average: {sum(data)/len(data):.2f}")
print(f"Maximum: {max(data)}")
print(f"Minimum: {min(data)}")

# Find outliers (like anomaly detection)
threshold = 50
outliers = [x for x in data if x > threshold]
print(f"Values above {threshold}: {outliers}")

Quick Quiz

Test Your Python Knowledge

Answer these questions to check your understanding of Python basics.

1. Which of these is a valid variable name in Python?

Correct Answer: first_name

Explanation: Python variable names can contain letters, numbers, and underscores, but cannot start with a number or contain special characters like @ or -.

2. What is the output of this code?

numbers = [1, 2, 3, 4, 5]
print(numbers[1:4])

Correct Answer: [2, 3, 4]

Explanation: Python slicing uses [start:stop] where start is inclusive and stop is exclusive. So [1:4] gives elements at indices 1, 2, and 3.

3. Which data structure removes duplicates automatically?

Correct Answer: Set

Explanation: Sets only store unique elements. If you try to add duplicate values, they are automatically removed.
