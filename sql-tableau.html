SQL & Tableau Topic Card

A responsive, interactive card component for learning platforms with progress tracking

Component Details

This implementation includes:

  • A responsive card layout with hover effects
  • Progress bar with dynamic width
  • Font Awesome icons for visual elements
  • Clean, modern design with gradient accents
  • Two card examples: one with 0% progress and one with 65% progress
  • Full HTML and CSS code available below

Note: The first card links to sql-tableau.html and shows 0% progress (Start Learning). The second card is a demo with 65% progress.
