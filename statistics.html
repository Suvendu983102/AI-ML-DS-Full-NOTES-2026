9 Measures of Central Tendency

Mean, Median, Mode

Central tendency measures describe the center point of a dataset. They are fundamental in understanding data distribution.

Mean (Average)

μ = Σx/n

The sum of all values divided by the number of values. Sensitive to outliers.

Median

Middle value

The middle value when data is sorted. Robust to outliers.

Mode

Most frequent value

The value that appears most frequently. Useful for categorical data.

Calculating Central Tendency in Python 
import numpy as np
from scipy import stats

# Sample dataset
data = [23, 29, 20, 32, 23, 21, 33, 25, 21, 30, 28, 23, 100]  # Contains outlier 100

# Mean
mean_value = np.mean(data)
print(f"Mean: {mean_value:.2f}")

# Median
median_value = np.median(data)
print(f"Median: {median_value:.2f}")

# Mode
mode_result = stats.mode(data)
print(f"Mode: {mode_result.mode[0]} (appears {mode_result.count[0]} times)")

# Trimmed mean (removes outliers)
from scipy.stats import trim_mean
trimmed_mean = trim_mean(data, proportiontocut=0.1)
print(f"Trimmed mean (10%): {trimmed_mean:.2f}")

# Weighted mean
weights = [1, 2, 3, 1, 2, 3, 1, 2, 3, 1, 2, 3, 0.1]  # Less weight to outlier
weighted_mean = np.average(data, weights=weights)
print(f"Weighted mean: {weighted_mean:.2f}")

When to Use Which Measure?

Situation Recommended Measure Reason
Normal distribution, no outliers Mean Uses all data points efficiently
Skewed distribution or outliers Median Robust against extreme values
Categorical data Mode Shows most common category
Data with measurement errors Trimmed Mean Removes extreme values

10 Measures of Dispersion

Range, Variance, Standard Deviation

Dispersion measures describe how spread out the data is. They complement central tendency measures.

Range

Difference between maximum and minimum values. Simple but sensitive to outliers.

Range = Max - Min

Variance (σ²)

Average of squared differences from the mean. Measures spread in squared units.

σ² = Σ(x - μ)² / n

Standard Deviation (σ)

Square root of variance. Measures spread in original units. Most commonly used.

σ = √Variance

Interquartile Range (IQR)

Range of middle 50% of data. Robust to outliers.

IQR = Q3 - Q1
Calculating Dispersion in Python 
import numpy as np
from scipy import stats

# Sample datasets
data_a = [23, 29, 20, 32, 23, 21, 33, 25, 21, 30, 28, 23]  # Low dispersion
data_b = [10, 50, 5, 45, 15, 55, 2, 48, 20, 40, 8, 52]     # High dispersion

print("Dataset A (Low dispersion):", data_a)
print("Dataset B (High dispersion):", data_b)

# Range
range_a = np.ptp(data_a)  # peak-to-peak
range_b = np.ptp(data_b)
print(f"\nRange - A: {range_a}, B: {range_b}")

# Variance
variance_a = np.var(data_a)
variance_b = np.var(data_b)
print(f"Variance - A: {variance_a:.2f}, B: {variance_b:.2f}")

# Standard Deviation
std_a = np.std(data_a)
std_b = np.std(data_b)
print(f"Standard Deviation - A: {std_a:.2f}, B: {std_b:.2f}")

# Interquartile Range (IQR)
q1_a, q3_a = np.percentile(data_a, [25, 75])
q1_b, q3_b = np.percentile(data_b, [25, 75])
iqr_a = q3_a - q1_a
iqr_b = q3_b - q1_b
print(f"IQR - A: {iqr_a:.2f}, B: {iqr_b:.2f}")

# Coefficient of Variation (relative dispersion)
cv_a = (std_a / np.mean(data_a)) * 100
cv_b = (std_b / np.mean(data_b)) * 100
print(f"Coefficient of Variation - A: {cv_a:.1f}%, B: {cv_b:.1f}%")

# Z-scores (standardization)
z_scores_a = stats.zscore(data_a)
print(f"\nZ-scores for Dataset A (first 5): {z_scores_a[:5].round(2)}")

11 Probability Basics

Fundamental Concepts

Probability quantifies uncertainty. It's the foundation of statistical inference and machine learning algorithms.

Probability Rules

  • 0 ≤ P(A) ≤ 1
  • P(Sample Space) = 1
  • P(A') = 1 - P(A)

Types of Events

  • Independent: P(A∩B) = P(A)×P(B)
  • Mutually Exclusive: P(A∪B) = P(A)+P(B)
  • Conditional: P(A|B) = P(A∩B)/P(B)

Bayes' Theorem

P(A|B) = P(B|A)×P(A)/P(B)

Updates probability based on new evidence

Probability Calculations in Python 
import numpy as np
from scipy import stats

# Example 1: Dice probability
print("=== Dice Probability ===")
# Probability of rolling a 6 on fair die
p_six = 1/6
print(f"P(rolling 6) = {p_six:.3f}")

# Probability of rolling even number
p_even = 3/6  # 2, 4, 6
print(f"P(even) = {p_even:.3f}")

# Probability of rolling 6 twice in a row
p_two_sixes = (1/6) * (1/6)
print(f"P(two sixes) = {p_two_sixes:.4f}")

# Example 2: Card deck probability
print("\n=== Card Deck Probability ===")
# Standard deck: 52 cards, 4 suits, 13 ranks
# Probability of drawing an Ace
p_ace = 4/52
print(f"P(Ace) = {p_ace:.3f}")

# Probability of drawing a Heart
p_heart = 13/52
print(f"P(Heart) = {p_heart:.3f}")

# Probability of drawing Ace of Hearts
p_ace_hearts = 1/52
print(f"P(Ace of Hearts) = {p_ace_hearts:.4f}")

# Example 3: Conditional probability
print("\n=== Conditional Probability ===")
# In a class of 100 students:
# 60 study ML, 40 study DL, 20 study both
p_ml = 60/100
p_dl = 40/100
p_both = 20/100

# Probability of studying DL given studying ML
p_dl_given_ml = p_both / p_ml
print(f"P(DL | ML) = {p_dl_given_ml:.3f}")

# Example 4: Bayes' Theorem
print("\n=== Bayes' Theorem ===")
# Medical test: Disease prevalence = 1%
# Test accuracy: 95% true positive, 90% true negative
p_disease = 0.01
p_positive_given_disease = 0.95
p_negative_given_no_disease = 0.90

# Calculate P(disease | positive)
p_positive = (p_positive_given_disease * p_disease + 
              (1 - p_negative_given_no_disease) * (1 - p_disease))
p_disease_given_positive = (p_positive_given_disease * p_disease) / p_positive
print(f"P(Disease | Positive test) = {p_disease_given_positive:.3f}")

Probability Simulation

Important Probability Rules

Addition Rule
P(A∪B) = P(A) + P(B) - P(A∩B)

For non-mutually exclusive events

Multiplication Rule
P(A∩B) = P(A) × P(B|A)

For dependent events

Law of Total Probability
P(B) = Σ P(B|Aᵢ)P(Aᵢ)

When events partition sample space

Bayes' Rule
P(A|B) = P(B|A)P(A)/P(B)

Updates beliefs with evidence

12 Probability Distributions

Common Distributions in ML

Probability distributions describe how probabilities are distributed over values of a random variable.

Normal (Gaussian) Distribution

N(μ, σ²)
Characteristics:
  • Bell-shaped curve
  • Symmetric about mean
  • 68-95-99.7 rule
  • Central Limit Theorem applies
Parameters:
  • μ = Mean (center)
  • σ = Standard deviation (spread)
Applications in ML:
  • Error terms in linear regression
  • Assumption in many statistical tests
  • Feature normalization
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# Generate normal distribution
mu, sigma = 0, 1  # Standard normal
data = np.random.normal(mu, sigma, 1000)

# Calculate PDF
x = np.linspace(-4, 4, 100)
pdf = stats.norm.pdf(x, mu, sigma)

# Calculate CDF
cdf = stats.norm.cdf(x, mu, sigma)

# Calculate probabilities
p_less_than_1 = stats.norm.cdf(1, mu, sigma)
p_between = stats.norm.cdf(1, mu, sigma) - stats.norm.cdf(-1, mu, sigma)

print(f"P(X < 1) = {p_less_than_1:.3f}")
print(f"P(-1 < X < 1) = {p_between:.3f}")

Binomial Distribution

B(n, p)
Characteristics:
  • Fixed number of trials (n)
  • Two outcomes: success/failure
  • Constant probability (p)
  • Independent trials
Parameters:
  • n = Number of trials
  • p = Probability of success
Applications in ML:
  • A/B testing
  • Classification accuracy
  • Quality control
from scipy import stats

# Binomial distribution parameters
n = 10  # trials
p = 0.5  # probability of success

# Calculate probabilities
for k in range(n+1):
    prob = stats.binom.pmf(k, n, p)
    print(f"P(X = {k}) = {prob:.4f}")

# Cumulative probability
prob_at_least_7 = 1 - stats.binom.cdf(6, n, p)
print(f"\nP(X ≥ 7) = {prob_at_least_7:.4f}")

# Expected value and variance
expected = n * p
variance = n * p * (1 - p)
print(f"Expected: {expected}")
print(f"Variance: {variance}")

Poisson Distribution

Pois(λ)
Characteristics:
  • Events in fixed interval
  • Events occur independently
  • Constant average rate (λ)
  • Discrete distribution
Parameters:
  • λ = Average rate of events
Applications in ML:
  • Count data modeling
  • Customer arrivals
  • Defect counts
from scipy import stats
import numpy as np

# Poisson distribution parameters
lambda_param = 3  # average rate

# Calculate probabilities
for k in range(10):
    prob = stats.poisson.pmf(k, lambda_param)
    print(f"P(X = {k}) = {prob:.4f}")

# Probability of more than 5 events
prob_more_than_5 = 1 - stats.poisson.cdf(5, lambda_param)
print(f"\nP(X > 5) = {prob_more_than_5:.4f}")

# Simulate Poisson process
events = stats.poisson.rvs(lambda_param, size=1000)
print(f"\nSimulated mean: {np.mean(events):.2f}")
print(f"Simulated variance: {np.var(events):.2f}")

13 Permutations & Combinations

Counting Principles

Permutations and combinations are fundamental for counting possibilities in probability and statistics.

Permutations

nPr = n!/(n-r)!

Order matters

Example: Arranging books on a shelf

Combinations

nCr = n!/[r!(n-r)!]

Order doesn't matter

Example: Choosing team members

Calculating Permutations and Combinations in Python 
import math
from itertools import permutations, combinations

# Example 1: Basic calculations
print("=== Basic Calculations ===")
n = 5
r = 3

# Permutations
permutations_count = math.perm(n, r)  # Python 3.8+
print(f"Permutations of {n} items taken {r} at a time:")
print(f"   Using math.perm: {permutations_count}")
print(f"   Using formula: {math.factorial(n) // math.factorial(n - r)}")

# Combinations
combinations_count = math.comb(n, r)  # Python 3.8+
print(f"\nCombinations of {n} items taken {r} at a time:")
print(f"   Using math.comb: {combinations_count}")
print(f"   Using formula: {math.factorial(n) // (math.factorial(r) * math.factorial(n - r))}")

# Example 2: Listing all possibilities
print("\n=== Listing All Possibilities ===")
items = ['A', 'B', 'C', 'D']

print("All permutations of 2 items:")
for perm in permutations(items, 2):
    print(f"  {perm}")

print("\nAll combinations of 2 items:")
for comb in combinations(items, 2):
    print(f"  {comb}")

# Example 3: Real-world problems
print("\n=== Real-world Problems ===")

# Problem 1: Password combinations
# 4-digit PIN with digits 0-9
digits = 10
pin_length = 4
pin_combinations = digits ** pin_length
print(f"4-digit PIN combinations: {pin_combinations:,}")

# Problem 2: Committee selection
# Choose 3 people from 10
people = 10
committee_size = 3
committee_combinations = math.comb(people, committee_size)
print(f"Ways to choose {committee_size} from {people}: {committee_combinations:,}")

# Problem 3: Arranging books
# 5 books on a shelf
books = 5
arrangements = math.factorial(books)
print(f"Ways to arrange {books} books: {arrangements:,}")

# Problem 4: Poker hands
# Number of 5-card poker hands from 52-card deck
poker_hands = math.comb(52, 5)
print(f"5-card poker hands: {poker_hands:,}")

# Probability of specific hand (e.g., royal flush)
royal_flushes = 4  # One for each suit
p_royal_flush = royal_flushes / poker_hands
print(f"P(Royal Flush) = {p_royal_flush:.10f}")

Hypothesis Testing in Python 
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# Example 1: One-sample t-test
print("=== One-sample t-test ===")
# Test if sample mean differs from population mean
sample = [23.4, 24.2, 25.1, 23.8, 24.5, 24.9, 23.7, 24.8, 25.2, 24.0]
population_mean = 24.5

# Perform t-test
t_stat, p_value = stats.ttest_1samp(sample, population_mean)
print(f"Sample mean: {np.mean(sample):.2f}")
print(f"Population mean (H₀): {population_mean}")
print(f"t-statistic: {t_stat:.3f}")
print(f"p-value: {p_value:.4f}")

if p_value < 0.05:
    print("Result: Reject H₀ (significant difference)")
else:
    print("Result: Fail to reject H₀ (no significant difference)")

# Example 2: Two-sample t-test
print("\n=== Two-sample t-test ===")
# Compare means of two independent samples
group_a = [85, 88, 92, 78, 90, 87, 89, 84, 91, 86]
group_b = [78, 82, 85, 79, 81, 84, 80, 83, 82, 79]

# Perform independent t-test
t_stat, p_value = stats.ttest_ind(group_a, group_b)
print(f"Group A mean: {np.mean(group_a):.2f}")
print(f"Group B mean: {np.mean(group_b):.2f}")
print(f"t-statistic: {t_stat:.3f}")
print(f"p-value: {p_value:.6f}")

# Example 3: Chi-square test
print("\n=== Chi-square test ===")
# Test independence between categorical variables
observed = np.array([[50, 30, 20],  # Group 1 preferences
                     [30, 40, 30],  # Group 2 preferences
                     [20, 30, 50]]) # Group 3 preferences

chi2, p_value, dof, expected = stats.chi2_contingency(observed)
print(f"Chi-square statistic: {chi2:.3f}")
print(f"p-value: {p_value:.4f}")
print(f"Degrees of freedom: {dof}")

# Example 4: ANOVA (Analysis of Variance)
print("\n=== One-way ANOVA ===")
# Compare means of three or more groups
group1 = [23, 25, 28, 22, 24]
group2 = [30, 32, 29, 31, 33]
group3 = [35, 37, 36, 34, 38]

f_stat, p_value = stats.f_oneway(group1, group2, group3)
print(f"F-statistic: {f_stat:.3f}")
print(f"p-value: {p_value:.6f}")

if p_value < 0.05:
    print("Result: Significant difference between group means")
    # Post-hoc test if ANOVA is significant
    from statsmodels.stats.multicomp import pairwise_tukeyhsd
    import pandas as pd
    
    data = group1 + group2 + group3
    groups = ['G1']*len(group1) + ['G2']*len(group2) + ['G3']*len(group3)
    
    tukey = pairwise_tukeyhsd(data, groups, alpha=0.05)
    print("\nTukey HSD Post-hoc test:")
    print(tukey)

Types of Statistical Tests

Test Type When to Use Assumptions Python Function
t-test (one-sample) Compare sample mean to population mean Normality, independence stats.ttest_1samp()
t-test (two-sample) Compare means of two independent groups Normality, equal variances stats.ttest_ind()
Paired t-test Compare means of two related groups Normality of differences stats.ttest_rel()

Understanding P-values and Confidence Intervals

P-value

Probability of observing the data (or more extreme) if the null hypothesis is true.

Interpretation:
  • p < 0.01: Very strong evidence against H₀
  • p < 0.05: Strong evidence against H₀
  • p < 0.10: Some evidence against H₀
  • p ≥ 0.10: Weak or no evidence against H₀
Confidence Interval

Range of values that likely contains the true population parameter.

95% CI Interpretation:

If we repeated the study 100 times, 95 of the confidence intervals would contain the true population parameter.

Type I & II Errors
Reject H₀ Fail to reject H₀
H₀ True Type I Error (α) Correct
H₀ False Correct (Power) Type II Error (β)
Effect Size

Magnitude of the difference, independent of sample size.

Common Measures:
  • Cohen's d (t-tests)
  • η² (ANOVA)
  • Odds Ratio (Chi-square)
  • Correlation Coefficient

Exercise 1: Descriptive Statistics

Easy

Given the following dataset of exam scores:

scores = [85, 92, 78, 90, 88, 76, 95, 89, 84, 91, 87, 83, 79, 94, 86]

Calculate:

  1. Mean, median, and mode
  2. Range, variance, and standard deviation
  3. Q1, Q3, and IQR
  4. Z-score for the score 85
  5. Identify any outliers using the 1.5×IQR rule
import numpy as np
from scipy import stats

scores = [85, 92, 78, 90, 88, 76, 95, 89, 84, 91, 87, 83, 79, 94, 86]

# 1. Central tendency
mean = np.mean(scores)
median = np.median(scores)
mode_result = stats.mode(scores)

print(f"1. Mean: {mean:.2f}")
print(f"   Median: {median}")
print(f"   Mode: {mode_result.mode[0]} (appears {mode_result.count[0]} times)")

# 2. Dispersion
range_val = np.ptp(scores)
variance = np.var(scores, ddof=1)  # sample variance
std_dev = np.std(scores, ddof=1)

print(f"\n2. Range: {range_val}")
print(f"   Variance: {variance:.2f}")
print(f"   Standard Deviation: {std_dev:.2f}")

# 3. Quartiles
q1 = np.percentile(scores, 25)
q3 = np.percentile(scores, 75)
iqr = q3 - q1

print(f"\n3. Q1: {q1:.2f}")
print(f"   Q3: {q3:.2f}")
print(f"   IQR: {iqr:.2f}")

# 4. Z-score for 85
z_score = (85 - mean) / std_dev
print(f"\n4. Z-score for 85: {z_score:.2f}")

# 5. Outliers using 1.5×IQR rule
lower_bound = q1 - 1.5 * iqr
upper_bound = q3 + 1.5 * iqr
outliers = [x for x in scores if x < lower_bound or x > upper_bound]

print(f"\n5. Lower bound: {lower_bound:.2f}")
print(f"   Upper bound: {upper_bound:.2f}")
print(f"   Outliers: {outliers if outliers else 'None'}")

Exercise 3: Hypothesis Testing

Hard

A company claims their new algorithm improves accuracy from the current 85%. They test it on 100 samples and get 92 correct predictions.

  1. Formulate null and alternative hypotheses
  2. Choose an appropriate statistical test
  3. Calculate the test statistic and p-value
  4. Interpret the results at α=0.05
  5. Calculate a 95% confidence interval for the true accuracy
  6. Discuss potential Type I and Type II errors
import numpy as np
from scipy import stats
import math

print("=== Hypothesis Testing Solution ===")

# Given data
n = 100
successes = 92
p_sample = successes / n
p_null = 0.85
alpha = 0.05

print(f"Sample size: {n}")
print(f"Successes: {successes}")
print(f"Sample proportion: {p_sample:.3f}")
print(f"Null hypothesis proportion: {p_null}")

# 1. Hypotheses
print("\n1. Hypotheses:")
print("   H₀: p = 0.85 (accuracy hasn't improved)")
print("   H₁: p > 0.85 (accuracy has improved)")

# 2. Test choice
print("\n2. Test:")
print("   One-sample proportion z-test")

# 3. Calculate test statistic and p-value
# z = (p̂ - p₀) / sqrt(p₀(1-p₀)/n)
se = math.sqrt(p_null * (1 - p_null) / n)
z_stat = (p_sample - p_null) / se
p_value = 1 - stats.norm.cdf(z_stat)  # One-tailed test

print(f"\n3. Calculations:")
print(f"   Standard Error: {se:.4f}")
print(f"   z-statistic: {z_stat:.3f}")
print(f"   p-value: {p_value:.6f}")

# 4. Interpretation
print("\n4. Interpretation:")
if p_value < alpha:
    print(f"   Reject H₀ (p-value {p_value:.4f} < α {alpha})")
    print("   There is significant evidence that the new algorithm improves accuracy")
else:
    print(f"   Fail to reject H₀ (p-value {p_value:.4f} ≥ α {alpha})")
    print("   No significant evidence that the new algorithm improves accuracy")

# 5. 95% Confidence Interval
# CI = p̂ ± z* × SE(p̂)
z_star = stats.norm.ppf(0.975)  # 97.5th percentile for 95% CI
se_ci = math.sqrt(p_sample * (1 - p_sample) / n)
ci_lower = p_sample - z_star * se_ci
ci_upper = p_sample + z_star * se_ci

print(f"\n5. 95% Confidence Interval:")
print(f"   CI = ({ci_lower:.3f}, {ci_upper:.3f})")
print(f"   We are 95% confident that the true accuracy is between {ci_lower:.1%} and {ci_upper:.1%}")

# 6. Error discussion
print("\n6. Potential Errors:")
print("   Type I Error (α = 0.05):")
print("   - Concluding algorithm improves when it doesn't")
print("   - Probability: 5% when H₀ is true")
print("\n   Type II Error (β):")
print("   - Failing to detect improvement when it exists")
print("   - Depends on effect size and sample size")
print("\n   Power (1-β):")
print("   - Probability of correctly detecting improvement")

Statistics Cheat Sheet

Descriptive Statistics
  • Mean: Σx/n
  • Median: Middle value
  • Mode: Most frequent
  • Variance: Σ(x-μ)²/n
  • Std Dev: √Variance
  • IQR: Q3 - Q1
Probability Rules
  • Addition: P(A∪B) = P(A) + P(B) - P(A∩B)
  • Multiplication: P(A∩B) = P(A) × P(B|A)
  • Complement: P(A') = 1 - P(A)
  • Conditional: P(A|B) = P(A∩B)/P(B)
  • Bayes': P(A|B) = P(B|A)P(A)/P(B)
Common Distributions
  • Normal: N(μ, σ²)
  • Binomial: B(n, p)
  • Poisson: Pois(λ)
  • Uniform: U(a, b)
  • Exponential: Exp(λ)
Hypothesis Testing
  • Significance: α = 0.05
  • Power: 1 - β
  • Type I Error: Reject true H₀
  • Type II Error: Fail to reject false H₀
  • p-value: P(data|H₀)
  • CI: Parameter ± margin

Pro Tip: Statistical Thinking

Master statistics through practice, not just theory:

  1. Always visualize data first - Create histograms, box plots, scatter plots
  2. Understand assumptions - Know when to use parametric vs non-parametric tests
  3. Focus on effect sizes - Statistical significance ≠ practical significance
  4. Consider multiple testing - Adjust p-values for multiple comparisons
  5. Validate with resampling - Use bootstrap for confidence intervals

Statistics is the language of data science. Invest time in mastering it!
